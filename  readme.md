###### 6.1 Activation FUnction
假设输入数据为二维，这里由于随机初始化w且每个w向量唯一对应二维平面内的一条直线，同时也可代表一个神经元（直线上方的区域表示该神经元的输出为正，下方为负，而为负的会导致回传梯度为0，即神经元无法更新）
[接上条]当数据云（即所有输入样本构成的集合）全部处于某个神经元对应直线的下方时，将导致无论输入哪个样本都无法使神经元的输出为正，因此该神经元在训练过程中将一直无法更新，所以说dead了
